Report a Bug

If you encounter an issue with Math Drills, we want to know so we can fix it quickly.

How to Report

  1. Email mathdrillsapp.petunia922@passinbox.com with the subject line “Math Drills Bug Report.”
  2. Include:
    • Your device type and iOS version.
    • The version of Math Drills you’re using (see the app’s settings screen).
    • A short description of the issue and any steps to reproduce it.
    • Screenshots or screen recordings if available.

We review every report and will follow up if we need more details. Thank you for helping us improve.
